WebGL Particle System

This was done as a university course project. The different attributes of the particles (position, color, size, ...) are controlled by expressions that the user controls, providing pretty nice flexibility. Those expressions can use a predefined set of variables, as well as all functions that the WebGL GLSL implementation supports.

Catmull-Rom splines

This is an implementation of the Catmull-Rom interpolation spline. Nowadays all image editing software that's worth anything comes with a Bezier curve editor of some kind, usually with an interface that lets the artist tweak the individual control points

The Catmull-Rom spline is unique amongst the family of Bezier-curve based splines, in that the control points are calculated automatically - the only input needed are the points that are going to be interpolated.
